# C++ Problem Solving

A structured C++ problem-solving repository containing matrix, string, and date-related challenges, along with a practical client management system.

This repository is built to strengthen logical thinking, algorithmic problem solving, and clean C++ implementation through organized, topic-based practice.

---

## Repository Overview

The project is divided into focused sections:

- **Matrix Problems**  
  Classic matrix operations and pattern-based problems.

- **String Problems**  
  String manipulation, parsing, formatting, and text-processing exercises.

- **Date Problems**  
  Date basics, arithmetic, comparison, utilities, working days, and periods.

- **Client System**  
  A console-based client management system that applies file handling and data conversion concepts in a real scenario.
